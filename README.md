# weathershell

This is Weather Shell repo for DevOps course

Script wet.sh will output current temperature and humidity in given city.
Information is requested by quering https://wttr.in


### Usage:

```
./wet.sh <CityName>
```


### Example:

```
./wet.sh Perm
City: Perm Temp: 23 C, Humidity: 47 %
```
